From Grasps to Dexterity:

Large-Scale Grasp Pretraining for Dexterous Manipulation

Anonymous Authors
Submission 351, CoRL 2026

Abstract

Large-scale dexterous grasp datasets provide rich priors for hand-object interaction, yet their use has largely been limited to grasp generation and pick-and-place manipulation. In this work, we study whether these priors can enable dexterous articulated tool use, which requires task-oriented functional dexterity. To address this problem, we adopt a hierarchical policy learning framework that combines high-level sub-goal prediction with a low-level goal-conditioned controller. We pretrain the low-level controller on large-scale grasp trajectories and fine-tune on downstream demonstrations. Further, we introduce DexCraft, a simulation benchmark with six articulated tool use tasks requiring fine-grained finger coordination. Experiments in simulation and the real world show that combining grasp pretraining with hierarchical policy learning improves the performance, suggesting that grasp synthesis datasets can be used as effective pretraining data for long-horizon dexterous manipulation.

DexCraft Benchmark

We propose the DexCraft benchmark which contains six dexterous manipulation tasks. In each task, the robot hand needs to grasp the articulated tool, lift it to a target pose and perform the function of the tool. We collect 5 demonstrations with human teleoperation and use MimicGen pipeline to augment to 500 demos.

How did we achieve this?


An overview of our approach. We integrate large-scale grasp pretraining with hierarchical policy framework. (a) High-level sub-goal prediction policy takes the current point cloud observation as input, and predicts the positions of hand key points. (b) Low-level policy is conditioned on predicted sub-goal key points and current observation, and predicts action chunks for the controller. We augment Dexonomy dataset into grasp trajectories and pretrain the low-level policy, then we fine-tune it with downstream task data.
